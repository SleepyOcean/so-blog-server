## 项目迁移



so-blog-server 现已迁移至新仓库 [so-server](https://github.com/SleepyOcean/so-server)。项目从单体架构，改为微服务的架构。目前so-server 已将文件服务、博客服务、权限服务等等拆分为单独子模块，在 so-server 仓库中统一管理。
